# AWS Foundation — Overview

Global infrastructure and core networking primitives you need before touching any AWS service.

## Concepts

- [Region](01_region.md) — Geographic areas where AWS runs infrastructure.
- [Availability Zone](02_availability_zone.md) — Isolated data centers within a Region.
- [Points of Presence](03_pop.md) — Edge locations for caching and acceleration.
- [Amazon VPC](04_amazon_vpc.md) — Virtual private network in AWS.
- [Subnet](05_subnet.md) — IP address ranges within a VPC.
