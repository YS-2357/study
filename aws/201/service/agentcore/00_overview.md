# AgentCore Services Overview

Amazon Bedrock AgentCore provides a set of composable services for deploying, operating, and governing AI agents in production, according to the [Amazon Bedrock AgentCore developer guide](https://docs.aws.amazon.com/bedrock-agentcore/latest/devguide/what-is-bedrock-agentcore.html).

## Study Order

1. [AgentCore Runtime](01_runtime.md) - Run agent code as managed serverless agent infrastructure.
2. [AgentCore Memory](02_memory.md) - Preserve useful context across agent sessions.
3. [AgentCore Gateway](03_gateway.md) - Expose APIs, Lambda functions, and MCP servers as agent tools.
4. [AgentCore Identity](04_identity.md) - Connect agents to AWS and external services without hard-coded credentials.
5. [AgentCore Code Interpreter](05_code_interpreter.md) - Execute generated code in an isolated sandbox.
6. [AgentCore Browser](06_browser.md) - Give agents managed browser automation.
7. [AgentCore Observability](07_observability.md) - Trace, debug, and monitor agent behavior.
8. [AgentCore Evaluations](08_evaluations.md) - Score agent quality from sessions and traces.
9. [AgentCore Policy](09_policy.md) - Enforce deterministic controls over tool use.
