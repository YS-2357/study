# aws/201
AWS 201 workspace for deeper architecture, implementation, and presentation work.

- `README.md`: Overview of the `aws/201/` folder and its direct contents.
- `notes/`: Study notes and topic breakdowns.
- `architecture/`: Architecture drafts, diagrams, and scenarios.
- `implementation/`: Build notes, code references, and implementation plans.
- `demo/`: Demo scripts, walkthroughs, and supporting assets.
- `ppt/`: Local-only presentation workspace.

## Local organizing rules

- `notes/` is for reusable topic research and planning notes.
- `architecture/` is for scenario design, diagrams, and system-level decisions.
- `implementation/` is for execution plans, technical breakdowns, and supporting build notes.
- `demo/` is for demo flows and reusable supporting artifacts.
- `ppt/` is for rehearsal and presentation prep only, so it stays local-only.

## Notes

- `ppt/` is kept as a local-only workspace and is intentionally ignored by Git.
