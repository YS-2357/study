# aws
AWS study materials grouped by level and purpose.

- `README.md`: Overview of the `aws/` folder and its direct contents.
- `101/`: AWS 101 notes, screenshots, console walkthroughs, and local presentation materials.
- `201/`: AWS 201 notes, architecture drafts, implementation plans, demo assets, and local presentation materials.
- `certificate/`: AWS certification study materials kept locally.

## Local organizing rules

- Organize AWS material by level or purpose, not by author, date, or temporary activity.
- Put foundational service understanding in `101/`.
- Put architecture, implementation, and demo-oriented work in `201/`.
- Keep certification-specific prep separate from reusable study material.

## Local-only areas

- `101/ppt/` is intentionally excluded from Git and reserved for rehearsal and presentation prep.
- `201/ppt/` is intentionally excluded from Git and reserved for rehearsal and presentation prep.
- `certificate/` is intentionally excluded from Git because it is private certification material, not reusable repo content.
