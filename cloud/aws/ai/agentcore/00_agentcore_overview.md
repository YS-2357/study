# AgentCore Services Overview

Amazon Bedrock AgentCore provides a set of composable services for deploying, operating, and governing AI agents in production, according to the [Amazon Bedrock AgentCore developer guide](https://docs.aws.amazon.com/bedrock-agentcore/latest/devguide/what-is-bedrock-agentcore.html).

## Concepts

- [AgentCore Runtime](01_runtime.md) - Run agent code as managed serverless agent infrastructure.
- [AgentCore Memory](02_memory.md) - Preserve useful context across agent sessions.
- [AgentCore Gateway](03_gateway.md) - Expose APIs, Lambda functions, and MCP servers as agent tools.
- [AgentCore Identity](04_identity.md) - Connect agents to AWS and external services without hard-coded credentials.
- [AgentCore Code Interpreter](05_code_interpreter.md) - Execute generated code in an isolated sandbox.
- [AgentCore Browser](06_browser.md) - Give agents managed browser automation.
- [AgentCore Observability](07_observability.md) - Trace, debug, and monitor agent behavior.
- [AgentCore Evaluations](08_evaluations.md) - Score agent quality from sessions and traces.
- [AgentCore Policy](09_policy.md) - Enforce deterministic controls over tool use.
