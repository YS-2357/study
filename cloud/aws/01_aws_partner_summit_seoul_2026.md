# AWS Partner Summit Seoul 2026

**Date:** April 21, 2026 | **Theme:** Year of Agents

## Key Numbers

- 2,300+ Korean AWS partners
- 11 AI Service Competency partners
- $370B APJ AI/GenAI spending forecast by 2029
- $200B AWS CapEx investment
- AWS Marketplace Korea: 90% YoY growth

## Core Themes

1. **Year of Agents** — 2026 declared the year of agentic AI; Bedrock + AgentCore + Strands is the core stack
2. **Every data opportunity is an AI opportunity** — data is the core differentiator (Rahul Pathak, AWS VP)
3. **Domain × AI = Partner value** — industry depth + AI differentiates MSPs from commodity cloud resellers
4. **Multi-tool workflows** — Transform → Kiro → DevOps Agent → circular reinforcement

## MSP Evolution

| Generation | Model | Focus |
|-----------|-------|-------|
| MSP 1.0 (On-Prem) | MONITOR | Break/fix, infra support, backup |
| MSP 2.0 (Cloud) | OPTIMIZE | ITIL, cloud coding, migration, security |
| MSP 3.0 (Cloud-Native) | INNOVATE | Modernization, DevSecOps/FinOps, AIOps |

## 4 Key Enablers (from 1,000+ customer conversations)

- **GOAL** — Focus on objectives that matter
- **DATA** — Unlock data for AI
- **GUARDRAILS** — Secure without stifling innovation
- **EXECUTION** — Sprint to outcomes in weeks, not months
