# AWS Database — Overview

Relational, NoSQL, in-memory cache, analytical, and search databases.

## Concepts

- [Amazon RDS](01_amazon_rds.md) — Managed relational databases.
- [Amazon Aurora](02_amazon_aurora.md) — High-performance relational database.
- [Amazon DynamoDB](03_amazon_dynamodb.md) — Serverless NoSQL database.
- [Amazon ElastiCache](04_amazon_elasticache.md) — In-memory caching.
- [Amazon Redshift](05_amazon_redshift.md) — Data warehouse.
- [Amazon OpenSearch](06_amazon_opensearch.md) — Search and analytics engine.
- [Amazon Neptune](07_amazon_neptune.md) — Managed graph database for knowledge graphs and GraphRAG.
