# Microsoft Copilot - Overview

Microsoft Copilot is Microsoft's AI assistant product family. It includes general chat, Microsoft 365 productivity assistants, developer tooling, business-agent builders, and security-focused assistants.

## Topics

*(Notes to be added)*

- Microsoft Copilot - general AI assistant for web, Windows, Edge, and search-style workflows
- Microsoft 365 Copilot - assistant inside Word, Excel, PowerPoint, Outlook, Teams, and other Microsoft 365 apps
- GitHub Copilot - coding assistant for IDEs, pull requests, chat, and developer workflows
- Copilot Studio - build and customize business Copilot agents
- Security Copilot - AI assistant for security operations
