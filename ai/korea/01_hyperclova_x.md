# HyperCLOVA X

## What It Is

NAVER's flagship Korean large language model. The largest Korean-language LLM built by a domestic company, trained on a Korean-dominant corpus. Deployed via CLOVA Studio (API platform) and powering NAVER's own services (search, shopping, maps).

## Topics

*(Notes to be added)*

- CLOVA Studio — API access, fine-tuning, prompt engineering tools
- HyperCLOVA X vs HyperCLOVA (original 2021 model)
- Enterprise use cases (government, finance, healthcare)
- NAVER Cloud integration
- Benchmark performance on Korean tasks vs GPT-4, Claude

## Why It Matters

HyperCLOVA X is Korea's most direct answer to GPT-4 — domestic, Korean-first, enterprise-grade. For any Korean enterprise that needs to keep data on domestic infrastructure, it's the primary alternative to US models.
